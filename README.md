【サービス名】
dazzling

【コンセプト】
舞台観劇を通して繋がる口コミサービス

【言語・フレームワーク】
・Ruby -ver 2.7.0 / Rails -ver 5.2.4.3

【開発環境】
ローカル(Rubymine -ver 2019-3)
mysql  Ver 8.0.17 for osx10.14 on x86_64 (Homebrew)

・ログイン/ログアウト機能(device)
・画像付き投稿機能(口コミ)
・投稿編集機能
・投稿一覧表示ページネーション機能(kaminari)
・同じ舞台に関する口コミをまとめて表示機能(未実装)
・同じ舞台への評価の平均点表示機能(未実装)

・投稿詳細表示機能(自分の投稿であれば編集可)
・投稿コメント機能
・星評価機能
・プロフィール編集機能(アイコン画像あり)
・いいね機能(Ajax 非同期処理)
・フォロー機能(Ajax　非同期処理)
・ダイレクトメッセージ機能
・キーワード検索機能
・都道府県絞り込み機能(Google map API)
・地図検索機能(Google map API)

【Rspec】
・単体テスト機能(現状は、user、post部分のみ)
・統合テスト機能
